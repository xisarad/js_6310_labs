# KAI Style Changer - Плагин для Chrome

Плагин для изменения стиля сайта kai.ru. Создан в рамках лабораторной работы №3.

## Функциональность

- Добавляет кнопку переключения стилей на страницы kai.ru
- Сохраняет состояние стиля между перезагрузками страницы
- Работает на главной странице и страницах навигации
- Изменяет более 8 различных стилей элементов

## Установка

1. Скачайте файлы плагина в папку `solutions/student-10/style-plugin/`
2. Откройте Chrome и перейдите на страницу `chrome://extensions/`
3. Включите "Режим разработчика" (переключатель в правом верхнем углу)
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами плагина

## Использование

1. Перейдите на сайт kai.ru
2. В правом верхнем углу появится кнопка "Стиль: ВЫКЛ"
3. Нажмите на кнопку для включения/выключения кастомного стиля
4. Состояние сохранится после перезагрузки страницы

## Изменяемые стили

- Цвет фона header и main content
- Стили карточек и новостных блоков
- Навигационное меню
- Цвета ссылок и кнопок
- Стили таблиц
- Заголовки различных уровней
- Отступы и границы элементов

## Технические детали

- Использует localStorage для сохранения состояния
- Работает через Content Scripts
- Совместим с Manifest V3
